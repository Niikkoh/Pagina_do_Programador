//JavaScript

Variaveis e constantes

//Tipos de variaveis declaradas no JavaScript.

  • let: Declara uma variável com escopo local;
  • var: Declara uma variável com escopo global (CAIU EM DESUSO);
  • const: Declara uma constante com escopo local.

Operadores aritméticos

//Operadores usados para cálculos matemáticos.

  • +: Soma;
  • -: Subtração;
  • *: Multiplicação;
  • >/: Divisão;
  • >%: Resto de divisão.
  • ++: Incrementa mais um;
  • --: Incrementa menos um.
  • +=: Atribuição com soma;
  • -=: Atribuição com subtração;
  • *=: Atribuição com multiplicação;
  • /=: Atribuição com divisão.
  • %=: Atribuição com resto de divisão.

Operadores de comparação

//Operadores usados para comparação de expressões.

  • ==: Compara se o valor é igual;
  • ===: Compara se o valor e o formato do conteúdo é igual;
  • !=: Compara se o valor é diferente;
  • !==: Compara se o valor ou o formato do conteúdo é diferente;
  • >: Maior que;
  • >=: Maior ou igual que;
  • <: Menor que;
  • <=: Menor ou igual que.

Operadores lógicos

//Operadores usados para lógica de comparações.

  • &&: Ambas as comparações devem ser verdadeiras;
  • ||: Uma das comparações deve ser verdadeira;
  • !: Negação da comparação

Estruturas de controle

//Síntaxe usada para manipulação das estruturas de controle.

  • Estruturas condicionais:
    • if ( [condição] ): Executa um bloco de comandos se a condição for verdadeira;
    • else if ( [condição] ): Executa um bloco de comandos se a condição for verdadeira e as anteriores vinculadas a essa falsa.
    • else: Executa um bloco de comandos se todas as condições anteriores que estão vinculadas a essa é falsa.
  • Estruturas de decisão:
    • switch ( [variavel] ): Define o escopo da estrutura switch e qual variável vai ser comparada;
    • case [valor]: Compara e executa um bloco de comandos se o valor definido for igual a variavel;
    • break: Quebra o escopo atual do código, nesse caso sai da estrutura switch.
    • default: Defini um bloco de comandos padrão caso a variável não seja igual a nenhum valor definido.
  • Estruturas de repetição:
    • for ( [variavel] = [valor inicial]; [condição]; [incrementação/decrementação da variavel] ): repete um bloco de comandos até a condição ser falsa, normalmente usada para laços definidos;
    • while ( [condição] ): repete um bloco de comandos até a condição ser falsa, normalmente usada para laços indefinidos;
    • do { [bloco de comandos] } while ( [condição] ): estrutura semelhante ao while, porém seu bloco de comandos é processado ao menos uma vez.

Funções

//Síntaxe usada para manipulação das funções.

  • function: Declara uma função;
  • return: Retorna um valor, ou objetdo, da função.

Classes e objetos

//Síntaxe usada para manipulação de classes e objetos.

  • class: Declara uma classe.
  • constructor( [variaveis] ): Método especial para declarar propriedades da classe.
  • this: Chama um método ou propriedade da classe no próprio escopo dela.
  • new: Instância um novo objeto.

Strings

//Síntaxe usada para Manipulação de strings

  • +: Contatena strings;
  • ${ [Variavel] }: Interpolação de strings.

Métodos e propriedades padrões para vetores

//Lista de métodos e propriedades usados para vetores na biblioteca padrão do JavaScript.

  • Métodos:
    • .shift(): Remove o primeiro elemento do vetor e retorna;
    • .pop(): Remove o ultimo elemento do vetor e retorna.
  • Propriedades:
    • .length: Informa o tamanho do vetor.

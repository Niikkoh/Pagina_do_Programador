JavaScript

Variaveis e constantes

Tipos de variaveis declaradas no JavaScript.

  • let: Declara uma variável com escopo local;
  • var: Declara uma variável com escopo global (CAIU EM DESUSO);
  • const: Declara uma contante com escopo local.

Operadores aritméticos

Operadores usados para cálculos matemáticos.

  • +: Soma;
  • -: Subtração;
  • *: Multiplicação;
  • /: Divisão;
  • %: Resto de divisão.
  • ++: Incrementa mais um;
  • --: Incrementa menos um.
  • +=: Atribuição com soma;
  • -=: Atribuição com subtração;
  • *=: Atribuição com multiplicação;
  • /=: Atribuição com divisão.
  • %=: Atribuição com resto de divisão.

Operadores de comparação

Operadores usados para comparação de expressões.

  • ==: Compara se o valor é igual;
  • ===: Compara se o valor e o formato do conteúdo é igual;
  • !=: Compara se o valor é diferente;
  • !==: Compara se o valor ou o formato do conteúdo é diferente;
  • >: Maior que;
  • >=: Maior ou igual que;
  • <: Menor que;
  • <=: Menor ou igual que.

Operadores lógicos

Operadores usados para lógica de comparações.

  • && : Ambas as comparações devem ser verdadeiras;
  • || : Uma das comparações deve ser verdadeira;
  • ! : Negação da comparação

Métodos e propriedades para vetores

Lista de métodos e propriedades usados para vetores na biblioteca padrão do JavaScript.

  • Métodos:
    • .shift(): Remove o primeiro elemento do vetor e retorna;
    • .pop(): Remove o ultimo elemento do vetor e retorna.
  • Propriedades:
    • .length: Informa o tamanho do vetor.

HTML5

Principais tags:

Tags mais comuns do HTML5.

  • <html>: O navegador interpreta seu conteúdo como HTML;
  • <head>: Informações que o navegador vai carregar antes de exibir pro usuário;
  • <body>: Conteúdo que vai aparecer ao usuário;
  • <title>: Título da página que aparece na aba do navegador.

Principais atributos de tags:

Atributos que podem ser integrados a qualquer tag.

  • id= "[nome do identificador]" : Identificador do elemento, usado para seu reconhecimento no JavaScript;
  • style= "[comando em CSS]" : Integrar comandos de CSS diretamente no elemento;
  • class= "[nome da classe]" : Padronizar todos os elementos com o mesmo nome da classe a um estilo CSS predefinido.

Tags de formatação:

Tags que envolvem formatação de texto.

  • <h1> ao <h6>: Título do texto da página, do nível 1 ao 6;
  • <p>: Define parágrafo do texto;
  • <br />: Quebra linha do texto;
  • <b>: Formata o texto para negrito;
  • <i>: Formata o texto para itálico;
  • <u>: Formata o texto para sublinhado;
  • <strong>: Formata o texto para negrito e o navegador entende como destaque;
  • <mark>: Adiciona um marca texto;
  • <sup>: Formata para um texto sub escrito;
  • <blockquote>: Abre margem no texto, para citações ou observações;
  • <font>: Troca a fonte de um texto.
    • face= " [nome da fonte1], [nome da fonte2]... ":Defini a fonte do texto;
    • color= " [declaração da cor] ":Troca a cor do texto.

Tags para listas ou tabelas:

Tags que envolvem implementação de listas ou tabelas.

  • <ol>: Define uma lista ordenada;
  • <ul>: Define uma lista não ordenada;
  • <li>: Define um item da lista.

Tags de formulário:

Tags usadas para preeenchimento de formulário / entrada de dados pelo usuário.

  • <form>: Define um formulário;
    • name= "[Nome do formulário]" : Nome do formulário, usado para seu reconhecimento no JavaScript;
    • action= "[Endereço do servidor de destino]" : Informa o servidor ao qual os dados serão enviados;
    • method= : Defini o método ao qual os dados serão enviados.
      • "GET": Os dados são enviados de forma exposta, pela URL;
      • "POST": Os dados são enviados de forma segura, por requisição.
    • target= : Define onde a informação será trafegada...
      • "_self" : Em uma mesma aba;
      • "_blank" : Em uma nova aba.
    • autocomplete= : Define se o navegador vai manter a função de salvar os dados do formulário ou não.
      • "on" : Função de salvar os dados se mantém;
      • "off" : Função de salvar os dados é desabilitada.
  • <input />: Gera uma campo de preenchimento de dados para o usuário interagir.
    • type= : Define o tipo de campo de preenchimento.
      • "text": Define como um campo de texto;
      • "number": Define como um campo numérico;
      • "password": Define como um campo para senha;
      • "checkbox": Define como um campo para marcação.
      • "radio": Define que apenas um de dois ou mais campos com o mesmo nome possa ser marcado;
      • "file": Define como um campo para enviar arquivos.
        • multiple: Permite enviar múltiplos arquivos.
      • "date": Define como um campo com calendário;
      • "range": Define como um campo de scroll.
      • "color": Define como um campo de cor;
      • "email": Define como um campo para e-mail;
      • "url": Define como um campo com URL;
      • "hidden": Define como um campo invisível ao usuário.
    • min=" [valor inteiro] ": Inteiro mínimo permitido para entrada;
    • max=" [valor inteiro] ": Inteiro máximo permitido para entrada;
    • value=" [valor inteiro] ": Define um valor padrão;
    • disabled: Desabilita o campo.
  • <button>: Gera um botão ao qual o usuário pode interagir.
    • type= : Define o tipo de botão.
      • "button": Gera um botão clicável.
      • "submit": Envia o formulário vinculado.
      • "reset": Reseta o formulário vinculado.
  • <select>: Gera uma caixa com opções predefinidas.
    • multiple : Permite que o usuário selecione múltiplas opções.
  • <option>: Defini como uma opção da caixa select.
    • value= "[Nome do valor]" : Nome da opção escolhida que vai para o sistema;
    • selected : Defini qual opção está selecionada.
  • <textarea>: Gera uma caixa de texto grande.
    • rows = " [numero natural] ": Defini o número de linhas que a caixa de texto vai exibir no layout.
    • cols = " [numero natural] ": Defini o número de colunas que a caixa de texto vai exibir no layout.
  • <label>: Representa a legenda para um item em uma interface de usuário.

Tags de mídia:

Tags usadas para manipulação de dados de mídia.

  • <a>: Gera um texto com link.
    • href= : Define o link de destino por...
      • "[Link]": Um link na internet.
      • "[Diretório]/[Nome do arquivo]": um arquivo HTML no diretório.
    • target= : Define que ao clicar no novo link...
      • "_self" : Irá abrir o novo link na mesma aba;
      • "_blank" : Irá abrir o novo link em uma nova aba.
    • title= "[Conteúdo no título do link]": Conteúdo que irá aparecer no título do link.
  • <img />: Gera uma imagem.
    • src= : Define a imagem a ser exibida...
      • "[Link da imagem]": Por um link.
      • "[Diretório]/[Nome do arquivo]": Por um arquivo no diretório.
    • width= "[Valor da largura]": Define a largura da imagem.

Atributos de evento em tags:

Atributos que acionam um comando em JavaScript ao ocorrer determinado evento.

  • onclick = "[Comando JavaScript]": Aciona ao clicar com o mouse;
  • onsubmit = "[Comando JavaScript]": Aciona ao enviar o formulário;
  • onmouseover = "[Comando JavaScript]": Aciona ao passar o mouse sobre.

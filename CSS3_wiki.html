Seletores CSS:

//Define quais elementos que serão estilizados.

  • Seletor por tipo: Busca elementos por uma tag HTML específica;
  • Seletor por ID (#): Busca elementos através do atributo "id";
  • Seletor por Classe (.): Busca elementos através do atributo "class";
  • Seletor Universal (*): Seleciona todos os elementos HTML;
  • Seletor de atributo ([atrib]): Seleciona elementos que possuem um atributo específico, podendo ser também o valor em específico.
    • [atrib = "nome"]: Seleciona elementos que possuem exatamente o mesmo nome no atributo.
    • [atrib ~= "nome"]: Seleciona elementos que possuem esse nome no atributo separado por espaço.
    • [atrib |= "nome"]: Seleciona elementos que possuem esse nome no atributo separado por hífen.
    • [atrib ^= "nome"]: Seleciona elementos que possuem o mesmo préfixo no atributo.
    • [atrib $= "nome"]: Seleciona elementos que possuem o mesmo sufixo no atributo.
    • [atrib *= "nome"]: Seleciona elementos que possuem a palavra em algum lugar no nome do atributo.

Combinadores e agrupador de seletores:

//Explicita o tipo de relação entre 2 ou mais seletores

  • Agrupador de seletores (,): As regras CSS serão aplicadas para todos os elementos com um dos seletores.
  • Elemento com um conjunto de seletores (sem espaço): As regras CSS serão aplicadas para os elementos que tenha ambos os seletores.
  • Combinador descendente (espaço): Aplica para todas os seletores decententes dos seletores especificados.
  • Combinador filho (>): Aplica para os seletores que são filhos diretos no caminho do seletor especificado.
  • Combinador irmão adjacente (+): Aplica para o seletor irmão adjacente ao seletor especificado.
  • Combinador irmão em geral (~): Aplica para todos os irmãos do seletor especificado

Propriedades de formatação de texto:

//Propriedas que envolvem a formatação de texto.

  • font-weight: - Grossura da fonte.
    • bold; - negrito.
  • text-transform: - Converte todo o texto.
    • uppercase; - em maiusculo.

Propriedades de cor:

//Propriedas que envolvem as cores do elemento.

  • background: - define o background do elemento.
    • [cor]; - pela cor.
  • color:[cor]; - define a cor do texto.
